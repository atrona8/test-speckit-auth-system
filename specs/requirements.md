# Auth System

## Objectif
Créer un système d'authentification.

## Fonctionnalités

REQ-001 : L'utilisateur peut s'inscrire avec email + mot de passe  
REQ-002 : L'utilisateur peut se connecter  
REQ-003 : L'utilisateur peut se déconnecter  
REQ-004 : Vérification des identifiants  
REQ-005 : Stockage sécurisé des mots de passe
